jQuery 개요

jQuery 란?

존 레식이라는 사람이 만들었음
js를 이용하여 라이브러리를 만들어 놓은 것
jQuery를 이용하여 복잡한 자바스크립트 구문을 짧고 편하게 작성이 가능함

jQuery의 장점

1) DOM 조작이 간단
2) AJAX 를 지원한다
3) 플러그인, api 지원해주는 곳이 많음

jQuery 라이브러리 연결 방법

1) 파일을 가져와서 사용
2) 링크(CDN 방식)를 이용하여 사용
